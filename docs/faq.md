# FAQ

## Preciso pagar ferramenta?

Não para o MVP. O fluxo usa Azure Cost Export, Storage, CSV e Power BI Desktop.

## Posso testar sem Azure?

Sim. Use `dashboard/sample-data/costs-sample.csv`.

## O PBIX completo está incluso?

No modelo planejado, o `.pbix` completo pode ser entregue separadamente como kit pago.
