# Pré-requisitos

## Ferramentas

- Azure CLI
- PowerShell 7+
- Power BI Desktop

## Permissões

| Operação | Permissão mínima |
|---|---|
| Ler custos | Billing Reader ou Cost Management Reader |
| Criar exports | Cost Management Contributor |
| Criar budgets | Cost Management Contributor |
| Baixar CSV do Storage | Storage Blob Data Reader |
| Aplicar tags | Contributor no Resource Group |
